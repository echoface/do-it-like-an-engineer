# gitbook markdown居中显示图片

很简单，so easy，直的不行来弯的，硬的不行来软的，明的不行来暗的😄

#### 利用表格让图片居中

* markdown中这是显示表格的，那我们就将图片放到中间，在做一点点小改动就可以居中了

  | 0:0 | 11 | 0:0 |
  | :--- | :--- | :--- |
  |  |  |  |

* 一点点小改动，编程下面这样，图片就完全居中了Y...

```
| 0:0 | img  | 2:0 |
| --  | :--: | --  |
```
